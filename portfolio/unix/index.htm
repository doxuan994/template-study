Ways to look at File

View the contents of a File on the screen

Name Command Lines Options after the contents displayed Meaning Examples
View the contents of a File more SPACE down one page more filename then when the contents display hit SPACE to down one page
more b up one page more filename then when the contents display hit b to up one page
more d down one-half page more filename then when the contents display hit d to down one-half page
more RETURN down one line more filename then when the contents display hit RETURN to down one line
more q down one line more filename then when the contents display hit q to quit

Name Command Lines Flags Examples Explanation

Show the end of output or a file tail -numbers tail -15 Show 15 lines at the end
