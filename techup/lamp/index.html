Installation


Step 1: Install php5


Open Terminal -> Copy and Paste the following command lines -> Hit Enter

[sudo] nano /etc/apache2/httpd.conf

Scroll down find php5 -> delete # before it -> control + O -> control + X

sudo apachectl restart

create an index.php file in cd /Library/WebServer/Documents/

Step 3: Install phpMyAdmin


Download phpMyAdmin -> https://www.phpmyadmin.net/ -> drop to the same folder as index.php -> change its name to phpMyAdmin.
The index.php is stored in the Documents folder with the folowing directory /Library/WebServer/Documents/

Open a new Terminal window and go to Documents directory. Use commmand line below if you do not understand my words:

$ cd /Library/WebServer/Documents/

Give the right to use phpMyAdmin:

$ sudo chmod o+x phpMyAdmin/

Then go to forward one more to phpMyAdmin directory, use commmand line below if you do not understand my words:

$ cd phpMyAdmin/

Create a file in the phpMyAdmin folder named config

You can easily do that by using this command line:

$ mkdir config

Give the right to use config file you just created, this step is not really important if you can open this file without this command line, then it is not a problem:

$ sudo chmod o+x config

Open Browser go to http://localhost/phpMyAdmin/config/

A message will display like this:

See Forbidden
You don't have permission to access /phpMyAdmin/config/ on this server.

Then go to http://localhost/phpMyAdmin/setup/

Add password then save.

Create new folder in your phpMyAdmin folder

$ sudo mkdir /var/mysql

Open Terminal and in phpMyAdmin directory

$ sudo ln -s /tmp/mysql.sock /var/mysql/mysql.sock

Note: phpMyAdmin is optional, you can use other MySQL UI software that helps you work with MySQL database. I prefer Sequel Pro. Download Sequel Pro here: https://www.sequelpro.com/

Lamp Stack Tips and Tricks


Userful Command Lines


To find the Apache version

$ httpd -v

Start, Stop and Restart Apache

$ sudo apachectl start
$ sudo apachectl stop
$ sudo apachectl -k restart

To check php version

$ php -v

Set up Apache web server Mac OS in your username folder not in WebServer


Create a new folder in your [username] folder of your Mac called Sites.

Add some text to username.conf file

Then Open your Terminal and copy + paste the following command line:

$ sudo nano /etc/apache2/users/[your-mac-user-name].conf

Terminal will display a file editor similarly to this:

GNU nano 2.0.6
File: /etc/apache2/users/[YourMacUserName].conf

Add the following lines to the script:

<Directory "/Users/username/Sites/"> AllowOverride All Options Indexes MultiViews FollowSymLinks Require all granted </Directory>

Note: Be sure to replace “username” with your actual username!

Control+o To Save
Control+x To Exit

Open httpd.conf and allow some modules:

1. Open httpd.conf

$ sudo nano /etc/apache2/httpd.conf

2. Allow all these lines -> delete # before it -> control + O -> control + X:


                            LoadModule authz_host_module libexec/apache2/mod_authz_host.so
                            LoadModule authz_core_module libexec/apache2/mod_authz_core.so
                            LoadModule userdir_module libexec/apache2/mod_userdir.so
                            LoadModule vhost_alias_module libexec/apache2/mod_vhost_alias.so
                            Include /private/etc/apache2/extra/httpd-userdir.conf
                            Include /private/etc/apache2/extra/httpd-vhosts.conf

Delete the # in front of these lines:


                            LoadModule include_module libexec/apache2/mod_include.so
                            LoadModule userdir_module libexec/apache2/mod_userdir.so
                            LoadModule rewrite_module libexec/apache2/mod_rewrite.so

Search for AllowOverride None -> AllowOverride All
LoadModule php5_module libexec/apache2/libphp5.so
Include /private/etc/apache2/extra/httpd-userdir.conf
Control+o To Save
Control+x To Exit

3. Edit httpd-userdir.conf

Open Terminal and Copy + Paste this to get to the file for editing:

$ sudo nano /etc/apache2/extra/httpd-userdir.conf

Uncomment the following line, removing #:

Include /private/etc/apache2/users/*.conf

Control+o To Save
Control+x To Exit
