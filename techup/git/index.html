Git

Step 1: Download git

Step 2: Checking Your Settings

If you want to check your settings, you can use the git config --list command to list all the settings Git can find at that point:

$ git config --list

Note: There are bunch of information you can find but focus on the user.name= and user.email=. If you can find them then you are good to go. Otherwise, keep reading!!!

Step 3: Config git Username and Email

Use the following command lines to config your git username and email

$ git config -—global user.name [username]
$ git config -—global user.email [email]

Tip: You can check specific user name or user email by following command lines:

$ git config user.name
$ git config user.email

Step 4: Initializing a Repository in an Existing Directory

Note: With the 3 steps above you are ready to use Git command lines. The following steps guides you how to use Git command lines.

1. Create a folder with command lines:

$ mkdir my_git_folder

2. Go to git folder, you just created with command line:

$ cd my_git_folder

3. To create a new git repository. Use command line:

$ git init

This command line above will initialize an empty Git repository in this directory /Users/username/git/.git/

Step 5: Make Your First Git Commit

1. Open text editor and create a simple index.html file and save it to the "my_git_folder" folder.

2. Open Terminal and in the my_git_folder directory use command line below to check status of the file you just create you properly could not see any changes because before to commit anything firstly we need to add file to git. 

$ git status or $ git s

3. To add new change to git. Use this command line:

$ git add .

Add all changes we made.

4. Commit Changes

$ git commit -m "Put your comments here"

5. Check Git history

$ git log

See if you can find your Git commit with the comment you just wrote.

Done! Congratulations! You just did your first Git commit!


Note: Because my_git_folder is initialied when I created it on step 4, so in this step I only need to add my changes.

To read offical documentation please visit Git website https://git-scm.com/

GitHub is a website where you can upload a copy of your Git repository. If you want know how to create a GitHub account visit GitHub website here https://github.com/.
Steps to create a GitHub account are very easy but learn more about GitHub click here GitHub

Guide to push a new project in GitHub

Step 1: Firstly, Commit

$ git commit -m "Comments"

Step 2: Adding a remote, Sets the new remote

$ git remote add origin [remote repository URL]

Step 3: Verifies the new remote URL

$ git remote -v

Step 4: Pushes the changes in your local repository up to the remote repository you specified as the origin

$ git push -u origin master
